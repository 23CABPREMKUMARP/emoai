# EMO AI - 3D Animated Landing Page

A premium, futuristic landing page for an AI Emotion Detection application. Built with React, Three.js, GSAP, and Tailwind CSS.

- **3D Hero Scene**: Immersive background with a morphing, distorting 3D orb using React Three Fiber.
